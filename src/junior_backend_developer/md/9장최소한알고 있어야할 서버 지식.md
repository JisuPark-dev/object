# 9장 최소한 알고 있어야 하는 서버 지식
os 설치, 설정, 관리해봤냐?
서버 os에 대한 경험이 없으면 어플리케이션 문제인지 os 문제인지 확인하는데 어려움을 겪는다. 

root 계정으로 os 로그인해서 프로세스 실행 시 프로세스 파일을 root 권한으로만 접근 가능한 곳에 만듬
그런데 일반 계정으로 로그인해서 프로세스 실행하니깐 실행을 못함. 

보통 root 계정은 인프라 담당자만 사용한다. 일반 개발자들은 일반 계정을 사용한다. 

### chmod 권한 할당
chmod 777 run.sh
chmod u+x run.sh
-rwxr--r--. 1 vagrant vagrant 28 11월 10 08:26 run.sh
